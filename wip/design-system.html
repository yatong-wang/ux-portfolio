08 / In-page Navigation

Sticky bar of section links. The current section can be highlighted by adding .active to the corresponding .link-nav (e.g. via a scroll-spy script).

Live example (this page uses it at the top)

Typography Colors Buttons In-page nav

Container: .nav-in-page
position: sticky; top: 0; z-index: 10; display: flex; flex-wrap: wrap; gap: 0.75rem; margin-bottom: 5rem; border-bottom: 1px solid var(--color-border); padding-bottom: 1.5rem; background-color: var(--color-background-dark);

Links: .link-nav on each <a href="#section-id">. Active state: .link-nav.active (primary color, bold; set via JS from scroll position).

Script: js/section-nav.js — runs only when nav.nav-in-page exists; toggles .active by scroll position. Optional data-offset="120" on the nav to override trigger offset (px from top).
