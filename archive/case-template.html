01 / Overview

The Challenge

The existing application suffered from low user retention due to a complex navigation structure and outdated visual language. Users struggled to find core features like transfer and budget tracking, resulting in a 15% drop-off rate during onboarding.

The Solution

A complete redesign focusing on simplification and accessibility. We introduced a dashboard-first approach, streamlined the transaction flow, and implemented a dark-mode-native UI system that aligns with modern financial app standards.

02 / Research

User Insights

We conducted 12 user interviews and analyzed over 500 support tickets. The data revealed three main pain points affecting the user experience.

search

Navigation Fatigue

Users took an average of 5 clicks to complete a simple transfer.

visibility_off

Hidden Features

40% of users didn't know the budgeting tool existed.

contrast

Visual Clutter

Information density was too high for mobile screens.

security

Trust Issues

Inconsistent branding reduced perceived security.
